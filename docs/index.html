Overview

simple_cli provides a clean, fluent API for parsing command-line arguments in Eiffel applications. It handles flags, options with values, positional arguments, subcommands, and automatic help generation.

Fluent API

Chain configuration calls for clean, readable option definitions.

Flags & Options

Boolean flags, string options, defaults, and required options.

Typed Access

Get values as strings, integers, or booleans with defaults.

Auto Help

Generates formatted help text from option definitions.

Quick Start

Installation

  1. Set the environment variable: 
    export SIMPLE_CLI=/path/to/simple_cli
  2. Add to your ECF file: 
    <library name="simple_cli" location="$SIMPLE_CLI/simple_cli.ecf"/>

Basic Usage

local
    cli: SIMPLE_CLI
do
    create cli.make
    cli.set_app_info ("myapp", "My Application", "1.0.0")

    -- Define options
    cli.add_flag ("v|verbose", "Enable verbose output")
    cli.add_option ("o|output", "Output file", "FILE")

    -- Parse and use
    cli.parse

    if cli.help_requested then
        cli.print_help
    elseif cli.has_errors then
        cli.print_errors
    else
        if cli.has_flag ("verbose") then
            print ("Verbose mode%N")
        end
        if attached cli.option_value ("output") as f then
            print ("Output: " + f + "%N")
        end
    end
end

Key Features

Configuration

  • set_app_info (name, desc, version) - Application metadata
  • add_flag (names, desc) - Boolean switch
  • add_option (names, desc, arg) - Option with value
  • add_option_with_default (names, desc, arg, default) - With default
  • add_required_option (names, desc, arg) - Required option

Name Formats

  • "v|verbose" - Short and long: -v or --verbose
  • "debug" - Long only: --debug
  • "d|dry-run" - Multi-word: -d or --dry-run

Command-Line Formats

  • --verbose - Long flag
  • -v - Short flag
  • -vdf - Combined short flags
  • --output=file.txt - Long option with =
  • --output file.txt - Long option with space
  • -o file.txt - Short option with space
  • -ofile.txt - Short option attached

Access

  • has_flag (name) - Check boolean flag
  • option_value (name) - Get string value
  • integer_option (name) - Get as integer
  • boolean_option (name) - Get as boolean
  • command - First positional argument
  • arguments - All positional arguments

Help Generation

  • help_text - Generate help string
  • print_help - Print to stdout
  • help_requested - Was -h/--help used?
  • version_requested - Was -V/--version used?

More Examples

Subcommand-Style CLI

local
    cli: SIMPLE_CLI
do
    create cli.make
    cli.set_app_info ("pkg", "Package Manager", "1.0.0")
    cli.add_flag ("v|verbose", "Verbose output")
    cli.parse

    if attached cli.command as cmd then
        inspect cmd
        when "install" then
            -- pkg install package1 package2
            across cli.arguments_after_command as pkg loop
                install_package (pkg)
            end
        when "remove" then
            -- pkg remove package1
            if cli.arguments_after_command.count > 0 then
                remove_package (cli.arguments_after_command.first)
            end
        else
            print ("Unknown command: " + cmd + "%N")
        end
    else
        cli.print_help
    end
end

Required Options

local
    cli: SIMPLE_CLI
do
    create cli.make
    cli.set_app_info ("convert", "File Converter", "1.0.0")
    cli.add_required_option ("i|input", "Input file", "FILE")
    cli.add_required_option ("o|output", "Output file", "FILE")
    cli.add_option_with_default ("f|format", "Output format", "FORMAT", "json")
    cli.parse

    if cli.has_errors then
        cli.print_errors
        cli.print_help
    else
        -- Required options guaranteed to have values
        if attached cli.option_value ("input") as inp and then
           attached cli.option_value ("output") as out then
            convert_file (inp, out, cli.option_value_or_default ("format", "json"))
        end
    end
end
